Key Shortcuts

    Files Tab
  • Ctrl + C: Copy cell content.
  • Ctrl + P: Paste cell content into another cell if it is editable.

    Video Tab

    Arena Tab

    Song Tab
  • P: Create a pulse
  • Del: Delete a pulse
  • Space: Next pulse
  • Right Arrow Key: Next pulse
  • Left Arrow Key: Previous pulse
  • Ctrl + C: Inside the "Statistics" tab the user can copy data form the statistics table.
