Song Tab

The song tab displays an audio file as a plot and provides detailed information on its data, which was obtained during pulse detection and statistical song analysis. The audio file can be played, paused and stopped. At the top right side the user can change settings regarding the data that is to be displayed, by choosing "Display Settings". The song data can be manually modified (creating and deleting pulses), saved and reloaded by choosing "Data Options". Also within the "Data Options" the user is able to perform a statistical analysis on the current song file.

General data about the song is displayed at the bottom of this view inside the "Song Details" tab. There are three lists. One listing all detected pulses and the corresponding cycles per pulse, one displaying the start positions of all IPIs, together with each IPI length in milliseconds and the final one listing all detected trains, the train lengths and the number of pulses inside a train. The user can click on each list entry to highlight the selected list item inside the plot or the other way round.

Inside the "Statistics" tab data that was calculated while running a statistical analysis is displayed, both as text and graphically.

Navigation:

  • Stop: If the song is played, it is stopped and the green cursor, displaying the current position, jumps to the songs beginning.
  • Play: Starts playing the song at the current position.
  • Pause: Pauses the song at the current position.
  • Previous: The green cursor jumps to the first previous pulse, that is not currently displayed.
  • Next: The green cursor jumps to the first next pulse, that is not currently displayed.
  • Sample Spinbox: The user can enter a sample number. The green cursor jumps to the entered position inside the song.
  • Time Spinbox: The user can enter a time and the green cursor jumps to the corresponding positon insie the song.

Display Settings:

  • Highlight Pulses: All detected pulses get highlighted in red.
  • Highlight Sines: All detected sines get highlighted in blue.
  • Highlight Trains: After pulse detection all putative pulse regions are highlighted in green. After running the statistical analysis detected trains are highlighted instead.

Data Options:

  • Show All: The plot is set to a zoom level where the whole song can be displayed inside the window.
  • Create Pulse: The user can create a pulse by selecting an area of the song plot by right clicking the right mouse button at the start position and dragging the mouse cursor to the desired end position, releasing the mouse button. After pressing the "Create Pulse" button, the selected area gets highlighted in red.
  • Delete Pulse: The user can delete one or several pulses by selecting them like described above and pressing the "Delete Pulse" Button. All pulses which are not fully selected, wont get deleted.
  • Statistical Analysis: The program performes a statistical analysis. The results can be viewed inside the Statistics Tab. Before running a statistical analysis the user can change some parameters choosing "Settings" -> "Song Analysis" -> "Song Statistics Options".
  • Save Data: All modified data gets saved.
  • Reload Data: Changes that were made without saving get undone.
