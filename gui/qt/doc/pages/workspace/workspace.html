Workspace

The MateBook workspace is devided into 5 views, which can be accessed by selecting a corresponding tab at the bottom of the program window. These five tabs are named:

  • Files: Lists all files that were added by the user.
  • Video: Video files can be displayed in this tab. Detected Arenas can be looked at and selected for a deailed view in the "Arena" tab.
  • Arena: Provides a detailed view of a selected arena.
  • Song: Song files can be displayed and modified.
  • Statistics: coming soon


See also: Files Tab, Video Tab, Arena Tab, Song Tab and Statistics Tab
