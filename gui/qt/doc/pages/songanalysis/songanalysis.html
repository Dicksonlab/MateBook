Song Analysis

The song analysis allows automated processing and statistical analysis of the courtship songs of the fruit fly, also known as Drosophila melanogaster. First the courtship song is analysed for pulses, by running the pulse detection process. After that the user can modify the song manually (adding and deleting pulses), or run a statistical analysis immediately.



The statistical song analysis can be performed for one file while inside the song tab or for multiple files when the file tab is active. Inside the song tab the user also gets some graphical representations of the calculated statistical data.
